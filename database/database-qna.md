> Q. MySQL을 사용하다가 인덱스를 사용한 조회쿼리를 날려보다가 어느순간 페이지를 몇개 안읽더라구요! 
혹시 MySQL 내부에서 기본적으로 쿼리도 캐싱을 하는걸까요?
> 
> A. 실제 내부에서 (물리메모리 70%까지도 설정 해서 쓰곤한다.) 메모리에 쿼리를 캐싱해두는 기능을 지원합니다. 그래서 자주 사용되는 쿼리문은 실제 데이터베이스 액세스를 하지 않고 결과를 반환합니다.
